# Asteroid Drift

A simple Asteroids-inspired arcade game built with Python and Pygame where you pilot a spaceship through drifting space rocks, trying to survive as long as possible.

## Description

Asteroid Drift is a lightweight 2D space shooter inspired by the classic *Asteroids* arcade game. The player controls a spaceship that can rotate, thrust forward, and shoot bullets to destroy incoming asteroids.

The game focuses on simple physics, momentum-based movement, and collision mechanics.

## Motivation

This project was built to practice:
- Game loops in Pygame
- Object-oriented programming in Python
- Collision detection
- Basic physics simulation (velocity + friction)
- Clean separation of game entities
